## hauhaubot

*A simple python discord bot which responds to messages sent in a discord server.*

- Responds with simple text / links based on a commands list file
- Users can add new commands to the commands list via messages in discord
- Bot will repost reddit video links with versions that can be viewed directly in discord
- Can provide timers for commmon timeframes in OSRS, ex: `_timer willow` will ping the discord user in 280 minutes, when their willow tree should be fully grown
